        <h1>Data Structure Simulator Application</h1>
        <div class="section">
            <h2>Overview</h2>
            <p>This application is a comprehensive simulator for various data structures. It provides a user-friendly menu that allows users to select and perform different operations on data structures such as Arrays, Linked Lists, Stacks, Queues, and Graphs.</p>
        </div>
        <div class="section">
            <h2>Features</h2>
            <ul>
                <li>Interactive Menu to choose different data structures</li>
                <li>Operations for each data structure, including:</li>
                <ul>
                    <li><strong>Array:</strong> Insertion, Deletion, Display, Selection Sort, Bubble Sort, Linear Sort</li>
                    <li><strong>Linked List:</strong> Insertion, Deletion, Display</li>
                    <li><strong>Stack:</strong> Push, Pop, Display</li>
                    <li><strong>Queue:</strong> Enqueue, Dequeue, Display</li>
                    <li><strong>Graph:</strong> Add Vertex, Add Edge, Display Adjacency List</li>
                </ul>
                <li>Clear and easy-to-understand user interface</li>
            </ul>
        </div>
        <div class="section">
            <h2>Installation</h2>
            <p>To install and run the Data Structure Simulator Application, follow these steps:</p>
            <ul>
                <li>Clone the repository: <code>git clone https://github.com/yourusername/datastructure-simulator.git</code></li>
                <li>Navigate to the project directory: <code>cd datastructure-simulator</code></li>
                <li>Compile the source code: <code>gcc -o simulator main.c</code></li>
                <li>Run the application: <code>./simulator</code></li>
            </ul>
        </div>
        <div class="section">
